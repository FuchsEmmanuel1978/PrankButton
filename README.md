# Mission Cupidon 💘

Petite application Android en Kotlin qui permet de créer une page HTML surprise personnalisée, avec :

- un titre personnalisable ;
- une question personnalisable ;
- un message final personnalisable ;
- une photo choisie depuis la galerie ;
- un bouton "Non" qui fuit ;
- un bouton "Oui" qui grossit ;
- une page HTML finale autonome avec l'image intégrée en Base64 ;
- partage du fichier HTML via WhatsApp.

## Installation

1. Ouvre le dossier `MissionCupidon` avec Android Studio.
2. Laisse Android Studio synchroniser Gradle.
3. Branche ton téléphone Android ou lance un émulateur.
4. Clique sur `Run`.

## Utilisation

1. Change le texte si tu veux.
2. Choisis une photo depuis ta galerie.
3. Clique sur `Créer et envoyer sur WhatsApp ❤️`.
4. WhatsApp s'ouvre avec le fichier HTML prêt à envoyer.

## Note

WhatsApp envoie le HTML comme document. La personne qui reçoit le fichier devra l'ouvrir avec Chrome, Samsung Internet, Firefox ou un autre navigateur.
